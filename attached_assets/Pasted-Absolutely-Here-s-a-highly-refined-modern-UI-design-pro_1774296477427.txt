### ✅ **AI CSP Scheduler – Modern UI Enhancement Prompt (Design-Only Upgrade)**

> **Prompt:**  
> Redesign the visual appearance of an AI-powered CSP Scheduler web interface with a clean, modern, and visually impressive aesthetic — suitable for a high-end startup product demo.  
>
> 🔹 **DO NOT alter** any component structure, DOM hierarchy, layout positioning, or functionality.  
> 🔹 **Preserve all existing logic**, interactions, and element placements exactly as-is.  
> 🔹 Apply **visual-only improvements** through CSS, typography, color, spacing, and micro-interactions.
>
> ---
>
> ### 🎨 **1. Theme & Color**
> - Maintain a **dark theme** with a rich, immersive feel.  
> - Replace pure black backgrounds (`#000`) with a soft dark charcoal: `#0f0f12`.  
> - Introduce subtle **gradient accents** using a modern **purple → pink → blue** palette:
>   - Primary gradient: `linear-gradient(135deg, #7c3aed 0%, #ec4899 50%, #3b82f6 100%)`  
>   - Use gradients sparingly on buttons, active states, and hover effects for depth and vibrancy.
>
> ---
>
> ### 🪟 **2. Cards (Subjects, Teachers, Rooms, Time Slots)**
> - Apply **glassmorphism** styling:
>   ```css
>   background: rgba(255, 255, 255, 0.05);
>   backdrop-filter: blur(10px);
>   -webkit-backdrop-filter: blur(10px);
>   border-radius: 16px;
>   padding: 16px;
>   box-shadow: 0 8px 30px rgba(0, 0, 0, 0.4);
>   border: 1px solid rgba(255, 255, 255, 0.1);
>   ```
> - Add **smooth transition**:
>   ```css
>   transition: transform 0.3s ease, box-shadow 0.3s ease;
>   &:hover {
>     transform: translateY(-4px);
>     box-shadow: 0 12px 40px rgba(0, 0, 0, 0.5);
>   }
>   ```
> - Ensure internal elements have consistent vertical spacing (`gap: 12px` or `margin-bottom: 12px`).
>
> ---
>
> ### 📝 **3. Input Fields**
> - Style all inputs (text, select, number) to be:
>   ```css
>   width: 100%;
>   padding: 12px;
>   background: rgba(255, 255, 255, 0.08);
>   border: 1px solid rgba(255, 255, 255, 0.1);
>   border-radius: 10px;
>   color: #ffffff;
>   font-size: 15px;
>   outline: none;
>   transition: box-shadow 0.2s ease, border-color 0.2s ease;
>   ```
> - On focus:
>   ```css
>   box-shadow: 0 0 0 2px rgba(124, 58, 237, 0.5);
>   border-color: #7c3aed;
>   ```
>
> ---
>
> ### ✍️ **4. Typography & Readability**
> - Use a modern sans-serif font stack:  
>   ```css
>   font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
>   ```
> - Increase base font size to `15px` minimum.  
> - Labels: `font-weight: 500`, `color: #e0e0e6`, `margin-bottom: 6px`, `display: block`.  
> - Improve line height: `line-height: 1.5`.  
> - Ensure full visibility of teacher names, subjects, rooms, and time slots — no truncation unless necessary (use `white-space: normal`).  
>
> ---
>
> ### 🎯 **5. Generate Schedule Button**
> - Make it stand out with:
>   ```css
>   background: linear-gradient(135deg, #7c3aed, #ec4899);
>   color: white;
>   border: none;
>   padding: 14px 24px;
>   border-radius: 10px;
>   font-weight: 600;
>   font-size: 16px;
>   cursor: pointer;
>   box-shadow: 0 6px 20px rgba(124, 58, 237, 0.3);
>   transition: all 0.2s ease;
>   text-transform: uppercase;
>   letter-spacing: 0.8px;
>   ```
> - Hover effect:
>   ```css
>   transform: scale(1.03);
>   box-shadow: 0 8px 25px rgba(124, 58, 237, 0.4);
>   ```
> - Active state: slight press effect (`transform: scale(0.98)`).
>
> ---
>
> ### 📏 **6. Spacing & Layout Flow**
> - Add consistent vertical rhythm:
>   - Between input groups: `margin-bottom: 16px`
>   - Between cards: `margin-bottom: 20px`
>   - Inside containers: use `gap: 16px` in flex/grid layouts
> - Avoid visual clutter; ensure breathing room around every interactive element.
>
> ---
>
> ### 🖼️ **7. Icons (Delete, Section, etc.)**
> - Align icons vertically with their associated text using `display: flex` and `align-items: center`.  
> - Style icons with:
>   ```css
>   color: #a1a1aa;
>   transition: color 0.2s ease, transform 0.2s ease;
>   &:hover {
>     color: #ec4899;
>     transform: scale(1.2);
>   }
>   cursor: pointer;
>   ```
> - Use modern icon set (e.g., Lucide or Heroicons) at `18px–20px`.
>
> ---
>
> ### 🔻 **8. Scrollbar Styling**
> - Style scrollbar for a sleek, native-like feel:
>   ```css
>   ::-webkit-scrollbar {
>     width: 8px;
>   }
>   ::-webkit-scrollbar-track {
>     background: #0f0f12;
>   }
>   ::-webkit-scrollbar-thumb {
>     background: #2d2d37;
>     border-radius: 4px;
>   }
>   ::-webkit-scrollbar-thumb:hover {
>     background: #4b4b5a;
>   }
>   ```
> - Ensure smooth scrolling:
>   ```css
>   scroll-behavior: smooth;
>   ```
>
> ---
>
> ### ✨ **9. Micro-Interactions & Polish**
> - Apply global transition:
>   ```css
>   *, button, input, card {
>     transition: all 0.2s cubic-bezier(0.25, 0.8, 0.25, 1);
>   }
>   ```
> - Subtle fade-in on load (if applicable): `opacity: 0 → 1` over `0.4s`.  
> - Smooth hover feedback on all interactive elements.
>
