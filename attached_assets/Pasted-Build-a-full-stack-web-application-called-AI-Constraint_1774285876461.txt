Build a full-stack web application called **“AI Constraint Solver for Real Life (CSP App)”** that solves real-world scheduling problems using Constraint Satisfaction Problem (CSP) techniques.

---

## 🎯 Objective

The application should generate valid schedules (e.g., college timetable) by satisfying multiple constraints using AI techniques.

---

## 🧠 Core AI Concepts (must implement)

* Model the problem as a CSP:

  * Variables (e.g., subjects/classes)
  * Domains (time slots)
  * Constraints (no overlap, teacher availability, room constraints)

* Implement:

  1. Backtracking Search
  2. Forward Checking
  3. Constraint Propagation
  4. Heuristics:

     * Minimum Remaining Values (MRV)
     * Degree Heuristic

---

## ⚙️ Functional Requirements

1. User inputs:

   * List of subjects
   * Teacher assignments
   * Available time slots
   * Room details

2. System builds CSP model:

   * Assign variables, domains, constraints

3. Apply CSP algorithms:

   * Backtracking with forward checking

4. Output:

   * Valid timetable (grid format)
